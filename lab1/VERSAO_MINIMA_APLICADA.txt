📊 Quando servidor anuncia:
   1. Recebe anúncio "TcpServer\ttcp://192.168.1.5:9000"
   2. Grava URI com timestamp atual
   3. Log: "Registered service: TcpServer -> tcp://... (timestamp: ...)"

📊 Quando servidor crashou:
   • Servidor PARA de anunciar
   • URI fica no registry COM timestamp antigo
   • Nunca é removido automaticamente
   • ℹ️  Programador pode ver timestamp e decidir

�� Quando cliente consulta:
   • knownUrisOf("TcpServer", 1) retorna TODOS os URIs
   • Incluindo servidores offline (com timestamps antigos)
   • Cliente recebe array de URIs
   • ℹ️  Cliente deve verificar se quer usar URIs antigos
